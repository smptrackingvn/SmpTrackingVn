Enable GitHub Pages
  1. Push the docs/ folder to your default branch (e.g. main).
  2. Open the repository on GitHub → Settings → Pages.
  3. Under Build and deployment, set Source to Deploy from a branch.
  4. Choose Branch: main, Folder: /docs → Save.
  5. After a few minutes, a URL like https://<username>.github.io/<repo>/privacy-policy.html will be live.
  6. Paste that privacy policy URL in Play Console → App content → Privacy policy.

Before publishing: replace the placeholder contact email in privacy-policy.html (Contact section).
